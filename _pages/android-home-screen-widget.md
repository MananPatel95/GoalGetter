# How to add a Goal widget on your Android Home Screen

Add Goal Getter widgets to your Android Home Screen to track your goals and habits at a glance.

## Step-by-step instructions:

### 1. Long press on your Home Screen
- Press and hold any empty area on your Android Home Screen
- Wait for the menu to appear

### 2. Tap "Widgets"
- Look for the "Widgets" option in the menu
- Tap it to open the widget gallery

### 3. Search for Goal Getter
- Scroll through the available widgets
- Find and select "Goal Getter"
- Or use the search function if available

### 4. Choose your widget size
- Select from small, medium, or large widget sizes
- Each size shows different information:
  - **Small**: Single goal progress
  - **Medium**: Multiple goals overview
  - **Large**: Detailed progress with charts

### 5. Add to Home Screen
- Drag the widget to your desired location
- Release to place it on your Home Screen
- Tap "Done" to finish

## Android widget features:

### Small Widget
- Shows one goal at a time
- Displays progress percentage
- Quick habit check-off

### Medium Widget
- Shows multiple goals
- Progress bars for each goal
- Streak counters

### Large Widget
- Comprehensive goal overview
- Detailed progress charts
- Multiple goal types (completion, countdown, abstinence)

## Customization options:

### Goal Selection
- Choose which goals to display
- Set update frequency
- Customize appearance

### Update Settings
- Real-time updates
- Manual refresh option
- Background app refresh

## Troubleshooting:

### Widget not updating?
- Check Background App Refresh is enabled
- Ensure Goal Getter has permission to update
- Try removing and re-adding the widget

### Can't find Goal Getter?
- Make sure the app is installed
- Check if you're running Android 5.0 or later
- Restart your Android device and try again

### Widget not appearing?
- Check if widgets are enabled in your launcher
- Try using a different launcher app
- Check if Goal Getter has widget permissions
